# Quick Start Guide - Morrix Labs Website

## 🚀 Get Started in 2 Minutes

### 1. Install & Run
```bash
npm install
npm run dev
```
Visit `http://localhost:3000`

### 2. Deploy
```bash
npm i -g vercel
vercel
```

That's it! Your site is live. 🎉

---

## 📝 Make Changes

### Edit Content
Open `lib/constants.ts` and update:
- Services
- Projects
- Pricing plans
- Process steps

### Edit Colors
Edit `tailwind.config.ts` colors section

### Edit Text
All page text is in `components/sections/*.tsx` files
